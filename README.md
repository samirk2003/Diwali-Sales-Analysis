# Diwali-Sales-Analysis
Performed data cleaning, exploratory data analysis (EDA), and visualization on the Diwali Sales dataset using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## 📌 Project Overview
This project analyzes Diwali sales data to identify customer purchasing behavior and business insights using Python.

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Data Cleaning
- Removed null values
- Dropped unnecessary columns
- Changed data types
- Cleaned duplicate records

## 📈 Exploratory Data Analysis
- Gender-wise analysis
- Age Group analysis
- State-wise sales
- Occupation analysis
- Product Category analysis
- Top 10 Sold Products

## 📌 Key Insights
- Married women aged 26–35 years contributed the highest sales.
- Customers from Uttar Pradesh, Maharashtra, and Karnataka spent the most.
- IT, Healthcare, and Aviation professionals were major buyers.
- Electronics and Food categories showed high demand.
